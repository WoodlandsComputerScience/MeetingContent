What is a Website?

A website is what you are looking at right now! To create a website, we need to use HTML and CSS. HTML (Hyper Text Markup Language) is used to describe the structure of a website. By using tags such as <head>, <body>, <header>, and <paragraph>, you can organize the inforation on your website! CSS (Cascading Style Sheets) lets you style your website, and change things such as background colour.

HTML

HTML is the foundation of every website. Without HTML, you wouldn't be able to display any elements at all.

HTML was created in 1993 by Tim Berners-Lee. Its invention allowed users of the Internet to navigate webpages using hyperlinks, which led users to other webpages. This interconnectedness was a new invention that was fundamental to the new web.

HTML has come a long way since then, and many new concepts have been introduced in HTML 5. Examples include new semantic tags, which makes web pages more machine-readable, and new ways to handle graphical content, such as video and audio tags.

CSS

In 1996, CSS was introduced, revolutionizing webpages by seperating the content from the presentation.

Before the creation of CSS, styling a website was very difficult. Each HTML element had to be styled seperately, wasting huge amounts of space. CSS was invented to solve this problem, using external stylesheets to act on a single document at once - "cascading".

In a new age of computation, standards for styling have increased drastically. CSS 3 introduces many new tools to enhance the look of your webpages. Examples include rgba, new drop shadow tools, new border style properties, and many new selectors.
